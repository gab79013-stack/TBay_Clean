## 🧹 About TBay Clean

TBay Clean is a premier commercial and residential cleaning service provider covering the entire San Francisco Bay Area. We deliver exceptional cleaning solutions tailored to meet the unique needs of businesses and homes across Northern California.

Inspired by industry leaders like Shine Facility Services, Cappstone, EH Clean, Dream Cleaning Company, and July's Commercial Cleaning, we bring professionalism, reliability, and attention to detail to every job.

## 🌟 Features

- **Modern Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **SEO Optimized**: Dedicated landing pages for 10 major Bay Area cities
- **Interactive UI**: Smooth animations, hover effects, and intuitive navigation
- **Service Showcase**: Comprehensive display of commercial, residential, and specialized cleaning services
- **Contact Integration**: Easy-to-use contact forms for quick service requests
- **Local Focus**: Tailored content for San Francisco, Oakland, San Jose, Berkeley, and more

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Optional: Local web server (e.g., Live Server for VS Code, Python http.server)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/gab79013-stack/TBay_Clean.git
   cd TBay_Clean
   ```

2. Open `index.html` in your web browser:
   - Double-click the file, or
   - Use a local server:
     ```bash
     # Using Python 3
     python3 -m http.server 8000
     
     # Then visit http://localhost:8000
     ```

## 🛠️ Customization

### Updating Content

1. **Text Content**: Edit HTML files directly to update service descriptions, about us, and contact info
2. **Images**: Replace Unsplash URLs in `index.html` and CSS with your actual business photos
3. **Contact Form**: Connect the form to your preferred backend service (Formspree, Netlify Forms, etc.)
4. **Phone/Email**: Update all instances of contact information throughout the site

### Adding More Cities

To add more city pages:
1. Copy an existing city HTML file from the `cities/` folder
2. Rename it to the new city (e.g., `palo-alto.html`)
3. Update the content, meta tags, and neighborhood lists
4. Add a link to the new page in the main navigation and services section
